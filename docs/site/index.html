Test REST routes without leaving WordPress

A free admin tool to list registered routes, send GET/POST/PUT/DELETE requests with custom headers and JSON bodies, and preview responses—optionally as different user roles.

Install from WordPress.org Developer documentation

What it does

Route discovery

Loads every route WordPress has registered, with HTTP methods. Filter by plugin or browse all.

Readable patterns

Complex regex route patterns are shown in a friendlier form so you can pick parameters quickly.

Full request control

Headers and body as JSON, form-style params, and role selection to test permissions.

Built for admins

Requires manage_options. Uses WordPress nonces and admin-ajax—no public endpoints added by the plugin.

Safety

This plugin is aimed at development and staging. It can create temporary users for role tests and invoke arbitrary REST routes your site exposes. Use with care on production.
