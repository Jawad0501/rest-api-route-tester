== Description ==

REST API Route Tester is a powerful tool for WordPress developers to test and debug REST API routes directly from the WordPress admin panel. This plugin provides a user-friendly interface to interact with your WordPress REST API endpoints, making development and testing more efficient.

## Key Features

- **Easy Route Testing**: Test any WordPress REST API route with a simple interface
- **Multiple HTTP Methods**: Support for GET, POST, PUT, and DELETE requests
- **Custom Headers**: Add custom headers to your API requests
- **Form Data Support**: Send form data with your requests
- **JSON Body Support**: Send JSON data in request body
- **Real-time Response**: View API responses instantly
- **Plugin-specific Routes**: Filter and test routes from specific plugins
- **Route Search**: Quickly find routes with the search functionality

## Perfect for Developers

Whether you're developing a new WordPress REST API endpoint or testing existing ones, WP REST Route Tester makes the process simple and efficient. The plugin provides all the tools you need to:

- Test API endpoints with different HTTP methods
- Send custom headers and data
- View detailed responses
- Debug API issues
- Filter routes by plugin
- Search for specific routes

## How to Use

1. Install and activate the plugin
2. Go to the REST Route Tester page in your WordPress admin panel
3. Select a route or enter a custom route
4. Choose the HTTP method
5. Add any necessary headers, form data, or JSON body
6. Click "Send" to test the route
7. View the response in real-time

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rest-api-route-tester` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->REST Route Tester screen to configure the plugin

== Frequently Asked Questions ==

= What is this plugin for? =

This plugin is designed for WordPress developers who need to test and debug REST API routes. It provides a user-friendly interface to interact with WordPress REST API endpoints.

= Do I need to be a developer to use this plugin? =

While the plugin is primarily designed for developers, anyone who needs to test WordPress REST API endpoints can use it. Basic knowledge of REST APIs and HTTP methods is helpful.

= Can I test routes from other plugins? =

Yes, you can test routes from any plugin that registers REST API endpoints. The plugin allows you to filter routes by specific plugins.

= Is this plugin safe to use on a production site? =

Yes, the plugin is safe to use on production sites. However, it's recommended to use it only when needed and to restrict access to users who need to test API routes.
